Instructions

  1. Enter your LinkedIn user id (stored in local storage).
  2. Compose your post and save it (stored in local storage).
  3. Click the "Copy HTML to Clipboard" button to copy the post content.
  4. Click the "Create a new LinkedIn post" link to open the LinkedIn post page.
  5. Open the Developer tools (F12), and run the following commands in the console:
    • html = `[paste from clipboard]`
    • $('.ql-editor').setHTMLUnsafe(html)
